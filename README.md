# Countdown
Dead simple SwiftUI countdowns app.

## Setup
### Download
1. Download and install the .ipa file from [releases](https://github.com/rrroyal/Countdown/releases).

### Build
1. Clone the repo.
2. Change every `please.change.me` occurrence to your Bundle ID.
3. Build and run.
